# 自动链接打开助手
项目仓库：QTMDJYFW

这是一个 Chrome 浏览器扩展，可以自动检测并打开链接，提高浏览效率。

## 功能特点

- 自动检测剪贴板中的 URL
- 在当前标签页打开链接，避免创建新标签页
- 支持开启/关闭功能
- 提供历史记录清理功能

## 安装方法

1. 下载本项目代码
2. 打开 Chrome 浏览器，进入扩展管理页面（chrome://extensions/）
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用方法

1. 安装扩展后，在 Chrome 浏览器右上角会出现扩展图标
2. 点击图标可以进行设置：
   - 启用/禁用扩展
   - 清除历史记录
3. 当复制一个链接时，扩展会自动在当前标签页打开该链接

## 注意事项

- 请确保给予扩展所需的权限
- 如遇到问题，可以尝试禁用后重新启用扩展
- 建议定期清理历史记录以保持良好性能
